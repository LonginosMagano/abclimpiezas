# ABC Limpiezas - Sitio Web Oficial

Sitio web profesional de **ABC Limpiezas**, empresa especializada en limpieza de incendios con más de 23 años de experiencia.

## Estructura del Proyecto

- **66 páginas HTML** con diseño moderno 2025
- **50 landings de provincias** con SEO programático
- **5 páginas de servicios** especializados
- **6 artículos de blog** sobre limpieza post-incendio
- **Páginas de contacto y legales**

## Tecnologías

- HTML5 semántico
- CSS3 moderno (variables, grid, flexbox)
- JavaScript vanilla
- Schema JSON-LD (LocalBusiness, FAQPage, BreadcrumbList)
- Mobile-first responsive design

## SEO

- Title tags optimizados (max 60 chars)
- Meta descriptions únicas por página
- Schema structured data
- Canonical URLs
- Open Graph tags
- Hreflang es-es
- .htaccess optimizado (HTTPS, gzip, caché)
